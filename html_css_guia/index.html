Estrutura Básica do HTML

Todo documento HTML segue uma estrutura fundamental que os navegadores usam para renderizar a página corretamente.

Anatomia de uma Página

  • <!DOCTYPE html> Declaração que define o tipo de documento.
  • <html> O elemento raiz que envolve todo o conteúdo.
  • <head> Contém metadados como título e links para CSS.
  • <body> Contém todo o conteúdo visível da página.

Exemplo de Código

<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <title>Título da Página</title>
    <link rel="stylesheet" href="style.css">
</head>
<body>
    <h1>Conteúdo Principal</h1>
</body>
</html>

Seletores CSS

Seletores são a base do CSS. Eles definem quais elementos HTML serão estilizados. Dominar seletores, desde os básicos (tipo, classe, ID) até os avançados (pseudo-classes, pseudo-elementos e combinadores), é essencial para um controle preciso.
